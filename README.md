🚀 JobHuntly – Applicant Tracking System (ATS)
JobHuntly is a modern, web-based full-stack Applicant Tracking System that streamlines the hiring workflow for both recruiters and job seekers. It integrates intelligent features like NLP-based skill analysis with secure, role-based access to create an efficient recruitment ecosystem.
✨ Overview
JobHuntly enables recruiters to manage job postings, evaluate candidates, and track applications, while applicants can explore opportunities, apply for roles, and monitor their progress—all on a single platform.
The system is built with scalability, security, and usability in mind, leveraging modern web technologies and backend-as-a-service architecture.
🧠 Key Features
🔐 Role-Based Authentication
Secure login/signup using Supabase Auth & JWT
Separate access for Recruiters and Applicants
Row Level Security (RLS) for fine-grained data protection
💼 Recruitment Management
Create, update, and manage job postings
Track candidate applications across different stages
View applicant profiles and resumes
👩‍💻 Applicant Experience
Browse and apply for jobs
Track real-time application status
Manage personal profile and uploaded resumes
📊 NLP-Based Skill Gap Analyzer
Compares applicant skills with job descriptions
Computes match percentage using NLP techniques
Identifies missing skills to guide applicants
📈 Dashboard & Workflow Design
Role-specific dashboards for recruiters and applicants
Organized workflows for hiring lifecycle management
